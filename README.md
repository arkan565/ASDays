# ASDays
##installation
Requeriments nodejs.
*Open terminal on the folder execute 
    *install dependencies with
    ```
    npm install
    ```
*after that you can execute the software
    *for development execute
    ```
    npm dev 
    ```
    *for next build execute
    ```
    npm build 
    ```
    *for production execute 
    ```
    npm start 
    ```
